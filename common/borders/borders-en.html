CSS Class

.brdr-left
Set a border to the left of the element
.brdr-tp
Set a border on top of the element
.brdr-rght
Set a border to the right of the element
.brdr-bttm
Set a border below the element
.brdr-0
Remove borders
.brdr-rds
Remove border radius
